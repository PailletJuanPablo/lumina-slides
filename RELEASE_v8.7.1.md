# Release v8.7.1

## Features

- **Speaker View Sync**: Real-time synchronization for multi-window speaker notes.
- **Firebase Core**: Integrated Firestore persistence and Cloud Functions for AI agent protocols.
- **Video Layout**: Dedicated layout for high-quality video content with custom controls.
- **New Themes**: Added `monochrome` and `sunset` presets.

## Improvements

- **Accessibility**: Enhanced contrast and safe-text enforcement across all layouts.
- **Dynamic UX**: Randomized background orb positioning for improved visual flow.
- **Deployment**: Refined `.npmignore` and project root configuration for cleaner distribution.

## Technical

- **Validation**: Strict Zod schemas for deck and theme configuration.
- **Integration**: Full Firebase SDK setup with environment variable support.
